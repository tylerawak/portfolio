Windows

Three options available — choose whichever is easiest for you.

Quick Assist

Built into Windows

Already installed on every Windows 10/11 PC — no download needed.

  1. Press Start, search Quick Assist, and open it.
  2. Click Get assistance.
  3. Enter the 6-digit code I send you and click Submit.
  4. Click Allow to share your screen.
Learn more about Quick Assist

HelpWire

Free download

A lightweight remote support tool — quick to install and easy to use.

  1. Download and run the HelpWire client from the link below.
  2. Share your ID and password with me.
  3. Click Allow when prompted to accept the connection.
Learn more about HelpWire

RustDesk

Free & open source

An open-source remote desktop tool — great if you prefer privacy-focused software.

  1. Download and install RustDesk from the link below.
  2. Share your ID and one-time password with me.
  3. Click Accept when the connection request appears.
